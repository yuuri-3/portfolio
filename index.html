How I work

良い所や存在意義を見つけます

ぱっと見で気付かなくても、人にも物にも必ず長所があり、作業にも意義があります。それを知らずにいるのはもったいない！と感じる性格です。

改善の余地を見出します

日頃から何のための作業か、何のための企画か、すべての「何のために」というゴールを見据え、到達するためにもっと良い方法や案はないかと思考を止めません。

戦略的縁の下

組織の動きや周りの空気に対して常にアンテナを張り、どうすれば良くなるか、そのために自分にできることは何なのか、縁の下的な動きをするのが好きです。

Skills

HTML5

最初に「HTML5/CSS3モダンコーディング」で基本的なHTMLの構造や必要な記述を学んだ後、その都度インターネットで情報を集めながらよりよい設計について学んでいます。

CSS3

最近のマイブームは、CSS設計に関する情報を読み漁ってもっともっとメンテナンスに強くて読みやすい設計を訓練していくことです。

Saas(SCSS)

とにかく導入してみよう！ということで、チュートリアルのみ学んだ後、Node.jsを使ったVScodeでの自動コンパイル設定を乗り越えて、やっと使い始めた所です。これから早くやりたいことは、パーツごとにファイルを切り分けてメンテナンス性をあげることです。

JavaScript

Progateで基本中の基本のみ学びましたが、普段は基本的に実装したい機能を調べる→コードを読んでそれぞれの行を理解する→次回につながるようにメモを残しながら実装する、というサイクルを繰り返しています。

jQuery

JSと同様で、使いたい機能基準でその都度まなんでいます。

Photoshop

Photoshop搭載のチュートリアルや、Adobe情報配信メールにあるテクニックなどを一通り試したあとは、ひたすら作品を作ることでデザインもツールも練習しています。直近は週に目標件数を決めてクラウドソーシングのコンペを題材にバナーを作成しています。

AdobeXD

一通りの機能は使えるようになりました。1つのデザインカンプを制作する際に新しい表現方法を1度か2度調べる程度です。これからやりたいことは、まだ知らない表現方法や機能の存在に気付くためにAdobeXDの参考書を読むことです（購入済み）。
